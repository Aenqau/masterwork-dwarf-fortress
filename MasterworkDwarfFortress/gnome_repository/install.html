Installation

This mod is pre-installed. Just download, unpack, thats it. When you are reading this, you already installed the mod, congratulation. Please do not copy it over vanilla DF or older mod versions. Just unpack it into a new folder.

To alter the mod, you can run the Masterwork Dwarf Fortress.exe. You will need to do this to set Gnomes as a playable race. Just open it up, click on 'Races' and you will see a large list of civilizations. Tick the box under 'Playable: Fortress' and all is done.

If you just want to play one race, it is recommended to turn the other races non-playable. It will make embarking easier, since you dont have to chose your race ingame.

How to do a bug report

Sorry, had to add this. When you report bugs, please state the mod version and the tileset you use. If you run the mod on Mac or Linux, please state this as well. I need this kind of info, otherwise I cant search in the right places. => Link to forum thread.
